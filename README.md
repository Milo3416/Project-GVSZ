# Project-GVSZ
Este es una landing que representa la comparativa entre dos SaaS con enfoques y objetivos de negocio diferente para compañias enterprise

# 🆚 GENESYS vs ZENDESK

## 🎯 POSICIONAMIENTO CLAVE

- **Zendesk** = Optimiza tickets  
- **Genesys** = Orquesta experiencias  

**Pitch:**  
> “No es una comparación de tools, es una decisión enfocada en la madurez y en la necesidad de evolucionar en Customer Experience”

---

## ⚖️ COMPARATIVO A GRAN ESCALA

| Categoría | Genesys Cloud CX | Zendesk | Insight clave |
|----------|-----------------|----------|--------------|
| ADN | Contact Center + Journey Orchestration | Ticketing + Soporte | Define el alcance real |
| Omnicanalidad | Nativa (voz + digital + bot + WEM) | Multicanal (bandejas unificadas) | Orquestar vs unificar |
| Voz | Core robusto (ACD, IVR, routing) | Limitado / partners | Crítico en enterprise |
| Routing | Predictivo (ML + intención + contexto) | Reglas y triggers | Inteligencia vs reglas |
| WEM | Completo (WFM, QM, coaching) | Limitado | Operación vs herramienta |
| Analítica | Journey + operación | Tickets | Experiencia vs soporte |
| Time to Value | Medio | Rápido | Quick wins vs estrategia |
| UX | Media | Muy intuitivo | Zendesk gana en facilidad |
| Escalabilidad | Enterprise (banca, telco) | Mid-market fuerte | Depende del caso |

---

## 🤖 ESTRATEGIA EN IA

### Enfoque

| | Genesys AI | Zendesk AI |
|--|-----------|------------|
| Tipo | Predictiva + Orquestación | Reactiva + Automatización |
| Alcance | End-to-end journey | Ticket / agente |
| Objetivo | Reducir contactos | Resolver contactos |

---

## ⚙️ CAPACIDADES DE IA

| Feature | Genesys | Zendesk | Insight |
|--------|--------|----------|--------|
| Bots | Avanzados + contexto omnicanal | Answer Bot | Similar base |
| Agent Assist | Tiempo real + next-best-action | Sugerencias | Genesys gana |
| Predictive Routing | IA (Machine Learning) | No equivalente | Genesys gana |
| Journey AI | Nativo | Limitado | Plataforma de CX real |
| Sentiment Analysis | Sí | Sí | Similar |
| Speech Analytics | Voz + texto | Limitado | Genesys clave en voice |
| Proactividad | Engagement predictivo | Reactivo | Antes vs después |

---

## 💡 MENSAJE DE IA CON DIFERENCIADOR

- Zendesk usa IA para resolver tickets más rápido  
- Genesys usa IA para mejorar la experiencia del cliente  

---

## 🧭 ¿CUÁNDO GANA CADA UNO?

### ✅ Zendesk gana cuando:
**Objetivo: Enfoque en tickets**
- Soporte simple / helpdesk  
- Implementación rápida  
- Equipos pequeños o medianos  

### 🔥 Genesys gana cuando:
**Objetivo: Estrategia de CX + automatización avanzada**
- Contact center complejo (voz + digital)  
- Necesidad de orquestación omnicanal real  
- Optimización de journeys  
- Uso de IA para impacto en negocio  

---

## 🎯 PREGUNTAS DE DISCOVERY O ACCIÓN

- ¿Qué porcentaje de contactos podrían evitarse?  
- ¿Cómo gestionan hoy la experiencia end-to-end?  
- ¿Miden resolución o experiencia?

- El cliente solo busca helpdesk  
- No hay visión de CX  
- Se compite en features en lugar de valor
